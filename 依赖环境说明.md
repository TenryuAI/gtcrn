# GTCRN 依赖环境说明

本文档说明项目所需的所有依赖环境，分为 **Python 环境**（用于模型推理、训练、ONNX 导出）和 **C++ 环境**（用于实时降噪程序编译运行）两部分。

---

## 一、Python 环境

### 基础信息

| 项目 | 版本 |
|------|------|
| Python | 3.11.x（64 位） |
| 操作系统 | Windows 10/11 x64 |

### Python 包依赖

#### 核心推理依赖

| 包名 | 版本（验证可用） | 用途 |
|------|----------------|------|
| `torch` (CPU) | 2.10.0 | 核心深度学习框架，模型推理、训练 |
| `numpy` | 1.26.4 | 数值计算 |
| `einops` | 0.8.2 | 张量重排（GTCRN 模型内部使用） |
| `soundfile` | 0.13.1 | WAV 音频文件读写（`infer.py`） |

#### 实时音频处理依赖

| 包名 | 版本（验证可用） | 用途 |
|------|----------------|------|
| `sounddevice` | 0.5.5 | 实时麦克风录音和耳机播放（`realtime_infer.py`） |

> `sounddevice` 底层依赖 **PortAudio**，Windows 下 pip 安装时会自动携带 PortAudio 动态库。

#### ONNX 导出依赖（仅导出模型时需要）

| 包名 | 版本（验证可用） | 用途 |
|------|----------------|------|
| `onnx` | 1.21.0 | ONNX 模型格式支持 |
| `onnxruntime` | 1.24.4 | ONNX 模型验证和 Python 推理测试 |
| `onnxsim` | 0.4.36 | ONNX 模型简化（减小模型体积，提升推理速度） |
| `onnxscript` | 0.6.2 | torch.onnx.export 依赖（PyTorch 2.9+ 自动安装） |

### 安装命令

```bash
# 安装 PyTorch（CPU 版本，适用于无 GPU 环境）
pip install torch --index-url https://download.pytorch.org/whl/cpu

# 安装核心依赖
pip install numpy einops soundfile sounddevice

# 安装 ONNX 导出相关依赖（仅需导出模型时安装）
pip install onnx onnxruntime onnxsim onnxscript
```

### 重要注意事项

#### 1. PyTorch 版本与 ONNX 导出器

- **PyTorch >= 2.9**：`torch.onnx.export` 默认使用新的 dynamo 导出器
- 新导出器存在 GRU 权重导出 Bug，导出时**必须**加 `dynamo=False` 参数
- 详见 `模型转换注意事项.md`

#### 2. Python ONNX Runtime 与 C++ ONNX Runtime 版本不一致

| 环境 | ONNX Runtime 版本 | 支持最高 IR version |
|------|------------------|--------------------|
| Python（导出验证） | 1.24.4 | IR version 10 |
| C++（实际推理） | 1.17.1 | IR version 9 |

Python 侧导出的 ONNX 模型 IR version 为 10，需要在导出后手动降级：
```python
import onnx
m = onnx.load('onnx_models/gtcrn_simple.onnx')
m.ir_version = 8
onnx.save(m, 'onnx_models/gtcrn_simple.onnx')
```

#### 3. 音频采样率要求

所有音频处理（Python 和 C++）均固定为 **16000 Hz（16kHz）**，因为模型在 DNS3 数据集（16kHz）上训练。输入音频必须为 16kHz，否则需要提前重采样。

## 二、C++ 环境

### 编译工具链

| 工具 | 版本要求 | 说明 |
|------|---------|------|
| **CMake** | >= 3.14 | 构建系统，自动管理依赖下载 |
| **Visual Studio** | 2019 / 2022（推荐） | 提供 MSVC 编译器（C++17），仅 Windows |
| **C++ 标准** | C++17 | `CMakeLists.txt` 中强制要求 |

### C++ 外部库（CMake 自动下载）

以下依赖库由 `CMakeLists.txt` 通过 `FetchContent` 自动下载，**无需手动安装**。

#### 1. ONNX Runtime 1.17.1（预编译二进制，Windows x64）

| 项目 | 说明 |
|------|------|
| 版本 | 1.17.1 |
| 下载地址 | `github.com/microsoft/onnxruntime/releases/download/v1.17.1/onnxruntime-win-x64-1.17.1.zip` |
| 用途 | C++ 推理引擎，加载并运行 `gtcrn_simple.onnx` |
| 运行时文件 | `onnxruntime.dll`（自动复制到 Release 目录） |
| 支持最高 ONNX IR version | 9（需要将导出模型 IR version 降级至 8） |

> **注意**：C++ 侧使用 1.17.1，Python 侧验证使用 1.24.4，两者版本不同。如需升级 C++ 侧 ONNX Runtime，修改 `CMakeLists.txt` 中的 `ORT_VERSION` 变量，同时确认与导出模型的 ONNX opset 版本兼容。

#### 2. PortAudio（从源码编译）

| 项目 | 说明 |
|------|------|
| 版本 | master 分支最新版 |
| 仓库地址 | `github.com/PortAudio/portaudio.git` |
| 用途 | 跨平台实时音频 I/O（麦克风输入 + 耳机输出） |
| 编译方式 | 静态库（不产生独立 DLL） |
| 运行时文件 | `portaudio.dll`（自动复制到 Release 目录） |
| 音频参数 | 16000 Hz、Float32、输入单声道、输出双声道、256 帧/块 |

#### 3. pocketfft（单头文件库）

| 项目 | 说明 |
|------|------|
| 版本 | cpp 分支 |
| 下载地址 | `raw.githubusercontent.com/mreineck/pocketfft/cpp/pocketfft_hdronly.h` |
| 用途 | 实现 STFT 和 iSTFT（实部/复数 FFT），仅需一个 `.h` 头文件 |
| 构建方式 | Interface Library（Header-only，无需编译） |

> **重要**：必须从 `cpp` 分支下载，`master` 分支不包含 `pocketfft_hdronly.h` 文件（第一次下载为空文件是 404 错误）。

### 编译步骤（Windows）

```bash
cd cpp_src
mkdir build && cd build

# 配置（自动下载所有依赖）
cmake ..

# 编译 Release 版本
cmake --build . --config Release
```

编译完成后，`Release/` 目录下包含以下所有文件，可直接运行：

```
Release/
├── gtcrn_realtime.exe    ← 可执行程序
├── gtcrn_simple.onnx     ← ONNX 模型（自动复制）
├── onnxruntime.dll        ← ONNX Runtime 运行时
└── portaudio.dll          ← PortAudio 运行时
```

### 运行命令

```bash
# 默认加载同目录下的 gtcrn_simple.onnx
cd cpp_src/build/Release
./gtcrn_realtime.exe

# 或指定模型路径
./gtcrn_realtime.exe "D:/github/gtcrn/onnx_models/gtcrn_simple.onnx"
```

## 三、模型文件说明

| 文件 | 大小 | 说明 |
|------|------|------|
| `checkpoints/model_trained_on_dns3.tar` | ~1MB | 原始 PyTorch 权重（BN 独立） |
| `onnx_models/gtcrn.onnx` | ~400KB | 导出的 ONNX 模型（未简化） |
| `onnx_models/gtcrn_simple.onnx` | ~560KB | 简化后的 ONNX 模型（供 C++ 使用） |

> `gtcrn_simple.onnx` 比 `gtcrn.onnx` 体积略大，是因为简化过程将常量折叠（constant folding）后的结果直接嵌入了模型。

## 四、整体依赖关系图

```
模型推理链路
─────────────────────────────────────────────────────
音频输入（WAV / 麦克风）
    │
    ▼
STFT（pocketfft / torch.fft）
    │
    ▼
GTCRN 流式推理（ONNX Runtime / PyTorch）
    │
    ▼
iSTFT + 重叠相加（Overlap-Add）
    │
    ▼
音频输出（WAV / 耳机）

Python 依赖                    C++ 依赖
─────────────────             ──────────────────────
torch 2.10.0                  ONNX Runtime 1.17.1
numpy 1.26.4                  PortAudio (master)
einops 0.8.2                  pocketfft (cpp branch)
soundfile 0.13.1              MSVC 2019/2022 (C++17)
sounddevice 0.5.5             CMake >= 3.14
onnx 1.21.0 (导出)
onnxruntime 1.24.4 (验证)
onnxsim 0.4.36 (导出)
```
